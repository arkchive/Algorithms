### 문제 요약
- 양의 정수 X가 주어질 때 분수열에서 X번째 분수를 구하는 문제
- 분수열은 대각선 단위로 나뉜다
    1층: 1/1
    2층: 1/2, 2/1
    3층: 3/1, 2/2, 1/3
    4층: 1/4, 2/3, 3/2, 4/1...

### 접근 & 오류
- 각 층의 길이는 층 번호와 같고, 층의 짝/홀에 따라 진행 방향이 결정된다.
- 분자와 분모를 구분해서 계산 
- 이전 층 마지막 인덱스와 분수가 층에서 몇 번째인지 (diff)를 구하지 않고 X를 바로 활용 -> 분자/분모 틀어짐 

### 해결 방법
- layer과 max를 통해 층을 찾는다. (while 루프 이용)
- 현재 층의 위치를 구한다. (prev, diff)
- 짝/홀 층 규칙 적용
    짝수 층 : 분자 증가, 분모 감소, 
    - num = diff
    - den = layer - diff + 1
    홀수 층 : 분모 증가, 분자 감소
    - num = layer - diff + 1
    - den = diff
